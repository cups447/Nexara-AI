# NEXARA AI — Institutional-Grade Crypto Trading Platform

An advanced AI-powered crypto trading signal platform that scans all Binance USDT Futures pairs using Smart Money Concepts (SMC) and 19+ technical indicators.

---

## Features

- AI confidence scoring system (only 90%+ signals sent)
- Smart Money Concepts: Order Blocks, FVG, BOS, CHoCH, Liquidity, Breaker Blocks
- 19+ Technical Indicators: EMA, RSI, MACD, ADX, Supertrend, Ichimoku, VWAP, and more
- Real-time scanning every 30 seconds across 8 timeframes
- Telegram bot with full command suite
- FastAPI REST API with authentication
- SQLite database with full CRUD
- Docker + Render deployment ready
